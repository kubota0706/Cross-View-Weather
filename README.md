# 気象予測AIプロジェクト

## プロジェクト概要
本プロジェクトは、地上設置のライブカメラ映像と気象衛星「ひまわり」の観測データを統合し、ディープラーニングを用いた**高精度な局所的気象予測**を行うシステムの開発を目的としています。

従来の広域な数値予報に加え、地上視点からの詳細な雲の挙動（形状・密度・進行方向）を解析することで、数十分から数時間先の天候変化をより具体的に予測するアプローチをとっています。

## データセットについて
本プロジェクトの学習用画像および各種データは、以下のオープンデータを利用して収集しています。

### 1. 地上カメラ画像（河川監視）
- **出典**: [国土交通省 近畿地方整備局 淀川河川事務所 ライブカメラ](https://www.kkr.mlit.go.jp/yodogawa/bousai/live.php)
- **取得元URL**: `https://cam.river.go.jp/`
- **利用目的**: 地上視点からの雲の形状・密度解析、および局所的な気象変化の学習

### 2. 気象衛星・解析データ
- **気象衛星「ひまわり」画像**: [気象庁 ナウキャスト](https://www.jma.go.jp/bosai/nowc/) および [JAXA ひまわりモニタ (P-Tree)](https://www.eorc.jaxa.jp/ptree/index_j.html)
- **出典元**: 気象庁 (JMA), 宇宙航空研究開発機構 (JAXA)
- **利用目的**: 広域の雲の動きの把握、および地上観測データとの突合による予測精度の向上

### 3. 気象観測データ
- **出典**: [気象庁 過去の気象データ検索](https://www.data.jma.go.jp/gmd/risk/strd/) (AMeDAS等)
- **利用目的**: 学習時の正解ラベル（アノテーション）の補完、および数値的な気象条件の統合

### 収集・利用ポリシー
- **サーバー負荷への配慮**: 対象サーバーへの過度な負荷を避けるため、各リクエスト間に1.5秒以上の間隔（`time.sleep`）を設けるスクレイピング・ポリシーを適用しています。
- **規約の遵守**: データの利用にあたっては、各機関の利用規約（[気象庁利用規約](https://www.jma.go.jp/jma/kishou/info/coment.html)等）を遵守し、学術・開発目的の範囲内で利用しています。

※本リポジトリには、学習済みのモデル（またはコード）のみを公開しており、著作権およびデータ容量の観点から、取得した画像データや数値データそのものは含んでいません。
